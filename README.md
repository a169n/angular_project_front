# Enterprise Angular Architecture

This repository contains a structured Angular front-end blueprint demonstrating advanced architecture, performance optimizations, and reactive data patterns.

## Features

- Modular architecture with `core`, `shared`, `features`, `state`, and `data-access` layers.
- Standalone components and lazy-loaded feature modules.
- Signals-based state management with computed selectors.
- RxJS retry, fallback, and centralized error handling.
- Light/dark mode theming with adaptive layouts.
